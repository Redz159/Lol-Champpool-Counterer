# Lol-Champpool-Counterer
Gets you an oversight of Champions, that counter multiple Champions at once, based on Lolalytics.com and your own configurations.

This workspace contains a starter scaffold (frontend + mock backend) for the web version of the app. The frontend uses React + Vite (JavaScript) and the mock backend is a small Express server that provides example endpoints for patches and counters.

Quickstart (PowerShell):

1. Frontend

```powershell
cd C:\\Users\\hendr\\IdeaProjects\\Lol-Champpool-Counterer
npm install
npm run dev
```

2. Backend

```powershell
cd C:\\Users\\hendr\\IdeaProjects\\Lol-Champpool-Counterer\\server
npm install
node index.js
```

The Vite frontend proxies `/api` requests to the mock backend on port 3000.
